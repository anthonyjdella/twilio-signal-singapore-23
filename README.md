## Setup

1. `cd Singapore-23`
2. `python3 -m venv .venv`
3. `source .venv/bin/activate`
4. `pip install Flask twilio python-dotenv pyngrok`
    OR
    `pip install -r requirements.txt`
5. `pip freeze > requirements.txt`
